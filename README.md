School Mangement System   
This system was developed from scratch and is still running with some modifications at the following url   
a. [School Management System](myschoolsystem.org)   
   
The system is pretty extensive and almost covers all the needs of a school that need to be systemized or digitalized   
1. Fees System
2. Exam System
3. Attendance System
4. Digital Diary
5. Notifications
6. Stakeholders Profiles
7. Revenue Calculations
8. Activity Logs
9. Staff Salary Management and much more.......

Instructions for usage:   
1. Simply clone the project
2. The DB structure is provided in Database folder, import it
3. For password hashing i am simply using md5
4. Create a client(school) in the database
5. Then create admin for that client and login

Note: These instructions are not enough for you to understand how to make this system operational, right now it will take a lot of time for me to update the readme, so if you figure this out thats good, In case not, i will be upgradind the instructions in near future.
